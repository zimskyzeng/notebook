# pyenv使用指引

## 0.安装pyenv及虚拟环境
brew install pyenv
brew install pyenv-virtualenv

## 1.查看可用版本
pyenv install --list | grep " 3\."

## 2.安装指定版本
pyenv install 3.9.18
pyenv install 3.11.5

## 3.切换python版本
### 全局默认版本
pyenv global 3.11.5

### 当前目录使用指定版本
pyenv local 3.9.18

### 临时使用版本
pyenv shell 3.11.5

## 4.查看本机已安装的版本
pyenv versions

## 5.新建virtualenv环境
pyenv virtualenv 3.9.6 myenv

## 6.激活使用virtualenv环境
pyenv activate myenv
